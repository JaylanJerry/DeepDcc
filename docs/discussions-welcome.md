# 欢迎来到 DeepDcc Discussions

这里是 DeepDcc 的公开讨论区，主要用于：

- 交流使用体验
- 讨论工作流和场景适配
- 收集测试版反馈
- 说明下载、更新与常见问题入口

## 先看 3 件事

1. 当前只公开 `testing` 通道  
2. 测试版采用“首次激活后滚动 30 天”  
3. 下载、FAQ、更新日志与反馈入口以官网为准：<https://deepdcc-pro.cn>

## 快速入口

- 官网主页：<https://deepdcc-pro.cn>
- 下载测试版：<https://deepdcc-pro.cn/downloads/DeepDcc-Test-Setup.exe>
- 更新日志：<https://deepdcc-pro.cn/changelog.html>
- 常见问题：<https://deepdcc-pro.cn/faq.html>
- 官网反馈页：<https://deepdcc-pro.cn/feedback.html>

## 什么问题适合发在这里

- 对产品定位、功能设计、工作流的看法
- 你更希望 DeepDcc 适配哪些文档场景
- 使用过程中的非敏感问题
- 对 README / 说明书 / 示例内容的建议

## 什么问题更适合走官网反馈页

- 安装失败
- 激活异常
- 导出错误
- 更新检测问题
- 带隐私、带资料、带具体客户信息的问题
