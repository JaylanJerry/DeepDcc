# 示例与适用场景

## 示例 1：教育分析报告

**输入方向**

> 生成一篇关于人工智能教育应用的分析报告，包含背景、核心场景、实施建议和结论。

**工作流亮点**

- 先出蓝图，再确认章节
- 长文生成后可继续在编辑器中调整
- 预览可用于核对结构和分页

配套示例：
- [教育分析报告示例](../assets/examples/education-report.md)

## 示例 2：公式型主题

**输入方向**

> 生成详细的薛定谔方程解析

**工作流亮点**

- 支持公式型内容进入正文链路
- 预览与正文需要保持一致
- 适合最终导向正式文档交付

配套示例：
- [公式型主题示例](../assets/examples/schrodinger-overview.md)

## 更适合的用户

- 需要正式报告而不只是聊天回答的用户
- 需要先确认结构，再释放正文的用户
- 需要公式、预览和正式导出链路一致的用户
- 对本地优先和失败可见更敏感的用户

## 暂时不主打的用户

- 只想快速问几句、拿一小段回答就结束的用户
- 不需要蓝图、预览、导出链路的用户
- 当前就要求公开正式版或团队协作后台的用户
