# 常见问题

## 现在开放的是正式版吗？

不是。当前对外只开放 `testing` 通道。

## 测试版能用多久？

当前测试版采用“首次激活后滚动 30 天”的策略。

## 更新是怎么检查的？

应用内 About 模块会从官网 `downloads/latest.json` 读取 testing 通道的更新信息。

## 为什么 GitHub 仓库里没有主开发源码？

这个公开仓库承担的是产品介绍、公开资料、下载导航和 Discussions 入口；主开发仓库不在这里公开维护。

## 遇到问题去哪里反馈？

优先顺序：

1. 官网反馈页
2. 本仓库 Discussions
3. 不适合公开的问题走私下渠道

## 为什么现在没有正式版下载？

因为当前仍处于测试版公开验证阶段，正式版与收费链路尚未公开上线。
