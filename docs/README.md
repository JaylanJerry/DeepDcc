## 你可以从这里得到什么

- DeepDcc 的定位、产品哲学与差异化
- 最值得关注的特色能力与能力边界
- 一条从输入任务到正式导出的完整工作流
- 从用户视角理解每一步该看什么
- 哪些场景更适合用 DeepDcc
- 为什么它更强调预览与导出结果
- 当前测试版、更新入口与反馈方式

## 公开仓库不包含什么

- 主开发源码
- 内部测试与发布脚本
- 内部修复记录与设计审查材料
- 正式版收费链实现细节

公开真源仍然以官网为准：<https://deepdcc-pro.cn>
